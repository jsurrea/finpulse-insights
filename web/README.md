# Usage

To set up the project, run:

```sh
npm install
```

For linting and formatting, run:

```sh
npm run lint
npm run format
```

For development, run:

```sh
npm run dev
```

For production, run:

```sh
npm run build
```

For unit testing, run:

```sh
npm run test:unit
```

For e2e testing, run:

```sh
npx playwright install
npm run test:e2e
```
